# Examples

This folder contains example reference images, generated panoramas, and notes from test runs.

## How to Add Examples

Create a separate folder for each example. Use clear names that describe the source scene or visual style.

Suggested folder structure:

```text
examples/
└── forest-path/
    ├── reference.jpg
    ├── panorama.jpg
    └── notes.md
```

## Suggested Notes Format

For each example, include:

- Reference image
- Generated panorama
- Prompt version
- Model used
- What worked
- What failed
- Notes for improvement

## Review Tips

Check each panorama in a viewer before judging it from the flat 2:1 image alone. Some problems only become obvious when the image wraps around the viewer.
